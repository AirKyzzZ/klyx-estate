# Klyx Estate

Marketing site for [Klyx Estate](https://estate.klyx.fr), a French studio producing cinematic listing films for prestige real estate on the Atlantic coast.

The site is the demo: the hero is a scroll-scrubbed film of a villa that was entirely generated from twelve stills by the studio's own pipeline, the same treatment a client mandate receives.

## Stack

- Next.js (App Router), TypeScript, Tailwind CSS v4
- Motion (framer-motion) + Lenis for the scroll choreography
- Custom scroll-scrubbed video hero, sticky horizontal filmstrip, cursor-follow offer previews
- Fraunces + Hanken Grotesk + Spline Sans Mono

## Run

```bash
npm install
npm run dev
```

Media assets in `public/media` are studio-generated and belong to Klyx Estate.
